# 幸运大转盘

一个美观、交互式的转盘，用于决策、抽奖和娱乐！自定义选项，转动转盘，享受惊艳的视觉效果。

## 功能特性

- **可自定义选项**: 添加、编辑、删除和重新排序转盘扇区
- **颜色选择器**: 为每个扇区选择自定义颜色
- **可调节旋转设置**: 控制旋转时长和力度
- **拖拽重新排序**: 轻松重新排列选项
- **视觉效果**:
  - 平滑的缓动旋转动画
  - 获胜时的彩纸庆祝效果
  - 动画结果弹窗
  - 指针脉冲动画
- **历史记录**: 查看最近的旋转结果
- **响应式设计**: 支持桌面和移动设备
- **美观的UI**: 现代渐变设计和玻璃态效果

## 使用方法

1. **添加选项**: 输入选项文本，点击"添加"或按回车键
2. **自定义颜色**: 使用颜色选择器更改下一个扇区的颜色
3. **重新排序**: 拖拽选项以改变它们在转盘上的顺序
4. **调整设置**: 使用滑块控制旋转时长和力度
5. **旋转**: 点击"旋转转盘"按钮，观看奇迹发生！

## 文件结构

- `index.html` - 主HTML文件
- `style.css` - 所有样式和动画
- `script.js` - 转盘逻辑和交互功能
